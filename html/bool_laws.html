Boolean Laws

Why Use Boolean Algebra
Boolean laws are a set of rules/laws that were standardised to help reduce the number of logic gates needed to perform a particular logic operation. These rules called theorems, are known commonly as the Laws of Boolean Algebra.

How Does Boolean Algebra work
The variables used in Boolean Algebra are bistable ie only 2 possible combinations: 0 or 1. However a boolean expression can have an infinite number of variables all labelled individually to represent inputs to the expression, For example, variables A, B, C etc, can give us a logical expression of A + B = C, but each variable can ONLY be a 0 or a 1.
