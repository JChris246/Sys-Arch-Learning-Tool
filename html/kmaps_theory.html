Karnaugh Maps

What are Karnaugh Maps
A Karnaugh Map is a graphical method of simplifying Boolean Algebra expressions. While Karnaugh Maps will produce the same simplified expression as using the Laws of Boolean Algebra, Karnaugh maps are more efficient as they reduce the need for extensive calculations by taking advantage of humans' pattern-recognition capability.

Steps to Construct a Karnaugh Map from an Expression


  1. Create a grid consisting of 2n squares, where n = number of unique input variables and label the variables used . (This varies depending on the number of variables used)

  2. The required inputs of the expression are transferred from a truth table onto the grid in the Karnaugh map. The cells are ordered in Gray code, and each cell position represents one combination of input conditions, while each cell value represents the corresponding output value. The Karnaugh Map is populated by placing a 1 in all the cells for which the output is 1, and placing a 0 in all the cells for which the output is 0.
    An X is placed in squares corresponding to don't-care conditions.
    (In digital logic, a don't-care term for a function is an input-sequence for which the function output does not matter.)

  3. Group adjacent 1’s in the largest possible groups of 2n (i.e. groups of 2, 4, 8 etc.) Aim to minimize the number of groups selected, while maximizing the number of elements within each group. After grouping, note the input combination of each group, opposite inputs will cancel. Variables within the same group are AND’d, however, when more than one group exists, the individual expressions from each respective group, are OR’d. This leaves you with the simplified expression of the original input.
